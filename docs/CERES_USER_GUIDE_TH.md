# คู่มือใช้งาน Ceres — ระบบบันทึกค่าใช้จ่าย (สำหรับทีม)

> เอกสารนี้เขียนเป็นภาษาไทยสำหรับแจกทีม (คัดลอกลง LINE หรือปริ้นต์ได้เลย)
> ระบบ Ceres = ตัวแทนดูแล "เงินออก" ของบริษัท (คู่กับ Juno ที่ดูแล "เงินเข้า")

**เว็บไซต์:** https://ceres-production-8d93.up.railway.app/
เปิดบนมือถือได้เลย แนะนำให้ "เพิ่มไปหน้าจอหลัก" (Add to Home Screen) จะใช้เหมือนแอป

**การเข้าระบบ (ทุกคน):** แตะชื่อตัวเอง → ใส่รหัส PIN 6 หลัก (หัวหน้าใช้รหัสผ่าน)
ระบบจะออกจากระบบเองทุก 12 ชั่วโมง ตอนเช้าให้เข้าใหม่

## 🛵 สำหรับพนักงานส่งของ (คนขับ / แมสเซนเจอร์)

**ทุกเช้า:** ไปรับเงินสดล่วงหน้าจากพี่นี (พี่นีจะเป็นคนบันทึกให้ในระบบ)

**ทุกครั้งที่จ่ายเงิน** (ค่าส่ง ค่าน้ำมัน ค่าทางด่วน ฯลฯ):
1. แตะปุ่ม **"บันทึกค่าใช้จ่าย"**
2. แตะ **"ถ่ายรูปใบเสร็จ"** แล้วถ่ายรูปบิล → ระบบ AI จะอ่านยอดเงินให้อัตโนมัติ
   *(ถ้าไม่มีบิลจริงๆ แตะ "ไม่มีใบเสร็จ" แล้วยืนยัน)*
3. **ตรวจยอดเงินให้ถูกต้อง** แล้วเลือก:
   - **PROM หรือ DENL** (บริษัทไหน)
   - **หมวดค่าใช้จ่าย** (เช่น ค่าน้ำมัน, ค่าขนส่ง SD)
   - ถ้าเป็น **ค่าขนส่ง** ให้ใส่ **ชื่อลูกค้า** ด้วย
4. แตะบันทึก — เสร็จ ✅

**ตอนเย็น:** คืนเงินทอน + บิลตัวจริงทั้งหมดให้พี่นี

**ความหมายของสถานะ:**
- 🟡 **รอตรวจ** = ยังรอพี่นีตรวจ (แก้ไข/ลบ ได้เฉพาะตอนนี้)
- 🟢 **อนุมัติแล้ว** = พี่นีอนุมัติเรียบร้อย
- ⚪ **ปิดยอดแล้ว** = ปิดยอดของวันนั้นแล้ว
- 🔴 **ตีกลับ** = มีปัญหา ให้ดูเหตุผล แล้วถ่ายบิลใหม่/แก้ไข

> 💡 บันทึกทันทีทุกครั้งที่จ่าย อย่ารอสะสมไว้ตอนเย็น จะลืมและยอดไม่ตรง

## 👩‍💼 สำหรับพี่นี (MD / หัวหน้า)

**ตั้งค่าครั้งแรกก่อนเริ่มใช้จริง:** นับเงินสดในกล่อง แล้วไปที่แท็บ **เบิก/คืน → ฝากเข้ากล่อง**
ใส่ยอดที่นับได้ (ทำครั้งเดียว)

**ตอนเช้า — จ่ายเงินล่วงหน้าให้พนักงาน:**
แท็บ **เบิก/คืน → เบิกเงิน** → เลือกชื่อพนักงาน + จำนวนเงิน → ยื่นเงินสดให้เขา

**ระหว่างวัน / ตอนเย็น — ตรวจและอนุมัติ:**
- แท็บ **รอตรวจ**: ดูรายการของพนักงาน + รูปบิล
  - ยอดตรง + มีบิล → แตะ **อนุมัติ**
  - ผิด/ไม่มีบิล → แตะ **ตีกลับ** แล้วพิมพ์เหตุผล
  - ⚠️ ป้ายเหลือง = ยอดที่พนักงานกรอกไม่ตรงกับที่ AI อ่านได้ ให้เช็คให้ดี
- แท็บ **กระดาน**: ดูช่อง **"เงินทอนที่ควรได้คืน"** ของแต่ละคน
  = ตัวเลขที่ใช้เทียบกับเงินสดจริงที่เขาคืนมา
- พนักงานคืนเงินทอน → บันทึกที่ **เบิก/คืน → รับเงินคืน**

**ปิดยอดสิ้นวัน:** แท็บ **ปิดยอด → "ปิดยอดวันนี้"**
> ต้องอนุมัติ (หรือตีกลับ) รายการ "รอตรวจ" ให้หมดก่อน ระบบถึงจะให้ปิด

**ลบ / ยกเลิกรายการ (แท็บ รายการ):**
- รายการที่ยัง **รอตรวจ** → ปุ่ม **ลบ** (ลบทิ้งได้เลย เพราะยังไม่ถูกนับ)
- รายการที่ **อนุมัติ / ปิดยอด / ตีกลับ ไปแล้ว** → ปุ่ม **ยกเลิก** (ต้องกรอกเหตุผล)
  รายการจะถูก **ตีเส้นทับ** ไว้ให้เห็น และ **ไม่ถูกนับในยอดใดๆ** อีก (กระดาน/ปิดยอด/รายงาน)
  แต่ระบบ **ไม่ลบทิ้งจริง** เพื่อให้ตรวจสอบย้อนหลังได้ และไม่ทำให้ยอดวันที่ปิดไปแล้วเพี้ยน

**จ่ายบิลบริษัท (เช่น ค่าไฟ ค่าของ):** แท็บ **จ่ายเงิน** → กรอกผู้รับ + จำนวน → AI ตรวจให้
- ≤ 5,000 บาท และผ่านนโยบาย → อนุมัติทันที จ่ายได้เลย
- \> 5,000 บาท หรือน่าสงสัย → ส่งให้ CEO อนุมัติก่อน (จ่ายไม่ได้จนกว่าจะอนุมัติ)
- จ่ายจริงแล้ว → กด **"จ่ายแล้ว"**

## 👔 สำหรับ CEO (คุณหมอ)

แท็บ **CEO** (มีเฉพาะคุณ):
- รายการที่รอคุณอนุมัติ (บิลเกิน 5,000 หรือที่ AI ส่งต่อ) → กด อนุมัติ / ปฏิเสธ
- ดูการตัดสินของ AI ทั้งหมดของวันนั้น พร้อมเหตุผล
- ดูเงินในกล่อง — ถ้าต่ำกว่า 40,000 บาท ระบบจะแนะนำยอดที่ควรเติม
- บิลที่เลยกำหนด และชุดไฟล์ CSV สำหรับตรวจสอบรายสัปดาห์

CEO ทำได้ทุกอย่างที่ MD ทำได้ รวมถึงการ **ลบ/ยกเลิกรายการ** ในแท็บ รายการ
