# Minerva OA Read Sync — ส่วนขยาย Chrome

ส่วนขยายเล็ก ๆ ที่ช่วยดึงสถานะ **"อ่านแล้ว" (Read / อ่านแล้ว)** จากหน้า **LINE OA Manager**
(`chat.line.biz`) เข้ามาแสดงใน **Minerva** โดยอัตโนมัติ — เพราะ LINE Messaging API ไม่ส่งสถานะ
"อ่านแล้ว" ออกมา สถานะนี้จึงมีอยู่แค่ในหน้า OA Manager เท่านั้น

ส่วนขยายนี้ทำงานแบบ **passive**: อ่านเฉพาะสิ่งที่แสดงอยู่บนหน้าจอของแชทที่คุณ *เปิดอยู่* คือ
**ชื่อ/รหัสแชท** และ **ป้าย "อ่านแล้ว"** เท่านั้น — **ไม่อ่านเนื้อหาข้อความ**, **ไม่เรียก API ลับของ LINE**,
และ **ไม่เก็บรหัสผ่าน/คุกกี้ของ LINE**

---

## วิธีติดตั้ง (ทำครั้งเดียว)

1. เปิด Chrome แล้วไปที่ **`chrome://extensions`**
2. เปิดสวิตช์ **"โหมดนักพัฒนา" (Developer mode)** ที่มุมขวาบน
3. คลิก **"โหลดส่วนขยายที่แตกไฟล์แล้ว" (Load unpacked)**
4. เลือกโฟลเดอร์ **`oa-sync-extension`** (โฟลเดอร์นี้)
5. จะเห็นไอคอน 👁 ของ **Minerva OA Read Sync** โผล่ขึ้นมาบนแถบส่วนขยาย

## วิธีเข้าสู่ระบบ (ทำครั้งเดียว)

1. คลิกไอคอน 👁 ของส่วนขยาย
2. ช่อง **เซิร์ฟเวอร์ Minerva** ปล่อยค่าเดิมไว้ได้เลย
   (`https://minerva-production-9309.up.railway.app`)
3. กรอก **อีเมล** และ **รหัสผ่าน** บัญชี Minerva ของคุณ แล้วกด **เข้าสู่ระบบ**
   - รหัสผ่านถูกส่งไปยัง Minerva เพื่อล็อกอินเท่านั้น **ไม่ถูกเก็บไว้** — เก็บเฉพาะ token
4. เมื่อขึ้นข้อความ **"ซิงก์ในนาม: (ชื่อคุณ)"** และสวิตช์ **เปิดใช้งานการซิงก์** เปิดอยู่ = พร้อมใช้งาน

## วิธีใช้งาน

ไม่ต้องทำอะไรเพิ่ม — แค่ **ใช้งาน LINE OA Manager ตามปกติ**
เมื่อคุณเปิดแชทลูกค้าคนไหน สถานะ "อ่านแล้ว" ของแชทนั้นจะถูกซิงก์เข้า Minerva อัตโนมัติ และจะไป
แสดงเป็นป้าย **"👁 อ่านแล้ว HH:MM"** บนหัวบทสนทนาของลูกค้าคนนั้นใน Minerva (เมื่อระบบจับคู่ลูกค้าได้)

> หมายเหตุ: สถานะจะซิงก์เฉพาะแชทที่ **มีคนเปิดดู** เท่านั้น — Minerva ไม่สามารถดึงสถานะของแชทที่ไม่มี
> ใครเปิดได้ ดังนั้นยิ่งทีมเปิด OA Manager ใช้งานปกติมากเท่าไร ข้อมูลก็ยิ่งครบเท่านั้น

## ปุ่มเปิด/ปิด

ในหน้าต่างส่วนขยายมีสวิตช์ **เปิดใช้งานการซิงก์** — ปิดเมื่อไรก็ได้เพื่อหยุดส่งข้อมูลชั่วคราว
โดยไม่ต้องออกจากระบบ

---

## ข้อจำกัด / ข้อควรทราบอย่างตรงไปตรงมา

- **LINE อาจปรับหน้าเว็บเมื่อไรก็ได้** — ถ้า LINE เปลี่ยนโครงสร้างหน้า OA Manager ส่วนขยายอาจอ่าน
  ป้าย "อ่านแล้ว" ไม่เจอชั่วคราว (จะไม่พังหรือรบกวนการทำงาน แค่หยุดซิงก์จนกว่าจะอัปเดต)
- **ซิงก์เฉพาะแชทที่เปิดดู** — ไม่ครอบคลุมทุกลูกค้าโดยอัตโนมัติ
- **การจับคู่ลูกค้าเป็นแบบระมัดระวัง** — OA Manager ใช้รหัสลูกค้าคนละชุดกับที่ Minerva เก็บ ระบบจึงจับคู่
  ให้เฉพาะเมื่อชื่อ **ตรงกันแบบไม่กำกวมเท่านั้น** ถ้าชื่อซ้ำ/ไม่ตรง จะเว้นว่างไว้ (ไม่เดา) — ถือเป็น "grey zone"
  ที่ยอมพลาดฝั่งไม่จับคู่ ดีกว่าจับผิดคน
- ส่วนขยายนี้เป็นเครื่องมือภายในทีม ไม่ได้อยู่บน Chrome Web Store — ติดตั้งแบบ Load unpacked ตามด้านบน
